жҰӮиҝ°

еҰӮдёҠж ҮйўҳпјҢзҺ°еңЁеҸҜд»ҘжӢҘжңүеұһдәҺиҮӘе·ұзҡ„дёӘдәәеҚҡе®ўдәҶ ~пјҲпјҫв–Ҫпјҫпјү~ пјҢ

е…ідәҺдёӘдәәеҚҡе®ўпјҢеҸҜд»ҘеҲ©з”Ё Github Page жқҘжҗӯе»әиҮӘе·ұзҡ„дёӘдәәеҚҡе®ўгҖӮ

жҖҺд№ҲеҒҡпјҹ

1. жіЁеҶҢ Github

ж—ўз„¶жҸҗеҲ°дәҶ Github Page пјҢйҰ–е…Ҳе°ұеҫ—жіЁеҶҢиҮӘе·ұзҡ„ Github иҙҰеҸ· гҖӮ

еҰӮжһң Github з•Ңйқўжү“дёҚејҖпјҢе»әи®®дҪҝз”Ё Watt Toolkit рҹҳғпјҢпјҲдёҚд»…еҸҜд»ҘеҠ йҖҹ SteamгҖҒж©ҷеӯҗгҖҒиӮІзў§пјҢеҗҢж—¶д№ҹиғҪеҠ йҖҹ Githubпјү

жіЁеҶҢдёҖдёӘ Github иҙҰеҸ·еҸӘйңҖиҰҒйӮ®з®ұпјҢе№¶дё”иҝӣиЎҢйӮ®з®ұйӘҢиҜҒе°ұиғҪеҝ«йҖҹе®ҢжҲҗпјҒгғҫ(вҖўПүвҖў`)o

2. еҲӣе»әиҮӘе·ұзҡ„д»“еә“

жіЁеҶҢд№ӢеҗҺпјҢжҺҘдёӢжқҘе°ұиҰҒеҲӣе»әиҮӘе·ұзҡ„д»“еә“пјҢ
йҰ–е…ҲзӮ№еҮ»гҖҗNewгҖ‘пјҢеҰӮдёӢеӣҫпјҡ

е…¶ж¬ЎпјҢеҲӣе»әд»“еә“ж—¶пјҢд»“еә“еҗҚдёҖе®ҡиҰҒд»Ҙ[дҪ зҡ„з”ЁжҲ·еҗҚ].github.io дёәеҗҚпјҢ й»ҳи®ӨдёӢйғҪдјҡе°Ҷе…¶и®ҫдёә Github PageпјҢд№ӢеҗҺе°ұиғҪе®ҢжҲҗеҲӣе»әгҖӮе…·дҪ“зңӢдёӢеӣҫпјҡ

еҲӣе»әе®ҢжҲҗд№ӢеҗҺпјҢзӮ№еҮ»иҝӣе…ҘдҪ зҡ„д»“еә“йЎ№зӣ®пјҢдҫқж¬ЎзӮ№еҮ»гҖҗSettingsгҖ‘вҶ’ гҖҗPagesгҖ‘вҶ’гҖҗDeploy from a branchгҖ‘пјҢе№¶йҖүжӢ©гҖҗGithub ActionгҖ‘пјҲдҪңжӯӨжӯҘжҳҜж–№дҫҝдёәдәҶеҗҺжңҹеҸҜд»ҘжӣҙжҚўжЎҶжһ¶жқҘж”№е–„иҮӘе·ұзҡ„еҚҡе®ўзҪ‘йЎөпјү

3. йҖүжӢ©жЎҶжһ¶

еҪ“дҪ еҒҡе®ҢдёҠиҝ°жӯҘйӘӨеҗҺпјҢдҪ еҸҜд»ҘзӣҙжҺҘи®ҝй—®иҮӘе·ұзҡ„зҪ‘йЎөпјҢеҰӮдёӢеӣҫпјҡ

жҺҘдёӢжқҘе°ұжҳҜйҖүжӢ©зҪ‘йЎөжЎҶжһ¶гҖӮиҝҷйҮҢе°ұд»Ҙ Hexo дёәдҫӢпјҢиҝҷйҮҢе°ұеҸӘйңҖиҰҒз”ЁеҲ° git е’Ң node.js дёӨдёӘдҫқиө–иҪҜд»¶дәҶпјҢе…ідәҺиҜҰз»Ҷең°е®үиЈ…жӯҘйӘӨпјҢиҜ·зӣҙжҺҘеҸӮз…§е®ҳж–№ж–ҮжЎЈ еҚіеҸҜ

е®ҢжҲҗеҗҺзҡ„жҗӯе»әпјҢеӨ§жҰӮжҳҜй•ҝиҝҷж ·зҡ„пјҡ

еҪ“з„¶пјҢиҝҷиҝҳдёҚеӨҹпјҢдҪ иҝҳеҸҜд»Ҙз»ҷе®ғжӣҙжҚўдёҖдёӘжӣҙеҘҪзңӢзҡ„дё»йўҳпјҢиҝҷйҮҢдҪңиҖ…е°ұз”ЁдәҶ Redefine дё»йўҳпјҢеӨ§жҰӮе°ұжҳҜй•ҝиҝҷж ·зҡ„(*^_^*)пјҡ

и·іиҪ¬жҸҗзӨә

е…ідәҺжҖҺд№Ҳй…ҚзҪ®иҝҷдёӘдё»йўҳпјҢ

иҜ·и·іиҪ¬еҲ°пјҡгҖҗжҗӯе»әдёӘдәәеҚҡе®ўпјҲдәҢпјүпјҡеҰӮдҪ•иҜҰз»Ҷй…ҚзҪ® Hexo дҪҝз”Ё Rendefine дё»йўҳзҫҺеҢ–еҚҡе®ўгҖ‘

жңҖеҗҺз®ҖеҚ•зҡ„еҒҡдёӘй…ҚзҪ®пјҢжңҖз»Ҳе‘ҲзҺ°дәҶиҝҷж ·зҡ„ж•Ҳжһңпјҡ

4. дёҠдј еҲ° Github

жҺҘдёӢжқҘпјҢе°Ҷжң¬ең°дёҠеҲ¶дҪңеҘҪзҡ„еҶ…е®№дёҠдј еҲ° Github д»“еә“дёӯгҖӮе…ідәҺжҖҺд№ҲдёҠдј йғЁзҪІ Github PagesпјҢиҜ·зӮ№еҮ»жҹҘзңӢе…ідәҺ Hexo дёҠдј йғЁзҪІеҲ° Github Pages зҡ„ж•ҷзЁӢ

йҰ–ж¬Ўжү§иЎҢпјҢйңҖиҰҒдёӢеҲ—зҡ„жүҖжңүе‘Ҫд»Өпјҡ

зӯүеҫ…дёҠдј е®ҢжҲҗеҚіеҸҜ(*^_^*)

еҶҷеңЁеҗҺйқў
